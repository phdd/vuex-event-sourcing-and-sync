# Vuex Mutation Event Sourcing and Synchronization

This is an Example on how to source events from mutations in vuex.
The running example has been done with the Todoist sync API.

## Project setup
Don't forget to add a Todoist API token as an environment variable to a file named`.env.local`

```properties
VUE_APP_TODOIST_TOKEN=yourpersonalaccesstoken
```

... and install
```shell script
npm install
```
